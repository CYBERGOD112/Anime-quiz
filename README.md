# JAMB-Style CBT Web App

## Overview
This is a **mini Computer-Based Test (CBT) web application** built using **Python** and **Flask**.  
It simulates a JAMB-style exam where users answer multiple-choice questions (A–D) one at a time, see their score at the end, and get a timestamp of when the test was submitted.  

This project demonstrates:
- **Object-Oriented Programming (OOP)** with the `Question` class  
- **Data Structures**: Queue logic to manage questions in order  
- **Standard APIs**: Python `datetime` module for timestamps  
- **Web Application** using Flask  
- **Professional UI**: Gradient background, centered card, box-shadow, and watermark  

---

## Features
- Start test from the homepage  
- Navigate through questions one at a time  
- Auto-calculates score  
- Displays submission time  
- Professional design with gradient background and centered quiz card  
- Watermark for branding
